Hawkins Labyrinth

Hawkins Labyrinth is a Stranger Things inspired adventure and survival game developed using HTML, CSS, and JavaScript. The game challenges players to navigate through a mysterious 5×5 labyrinth while managing their health, overcoming hidden obstacles, and escaping before Vecna becomes too powerful.

Every cell in the labyrinth contains a hidden unlock requirement. Players must roll a dice before each move, and the dice value must be greater than or equal to the cell's requirement in order to enter it. As the journey progresses, players encounter both safe Real World cells and dangerous Upside Down cells that can reduce health.

To increase the challenge, Vecna's health starts increasing continuously after the player's first successful move. The player's objective is to locate the hidden Exit Door and escape the labyrinth before Vecna's strength surpasses their own.

Gameplay Features

- Interactive 5×5 labyrinth board
- Dice-based movement mechanics
- Hidden unlock requirements for every cell
- Real World and Upside Down environments
- Health management system
- Dynamic Vecna health progression
- Randomized game board generation
- Secret Exit Door placement
- Win and Lose conditions
- Visual exploration tracking

Game Rules

- The player begins at Cell 21.
- A dice must be rolled before every movement.
- Movement is restricted to adjacent cells only.
- Every cell contains a hidden unlock value between 1 and 10.
- The dice value must be greater than or equal to the unlock value to enter a cell.
- If the requirement is not satisfied, the player loses the move and must roll again.
- Real World cells are safe and do not affect player health.
- Upside Down cells reduce player health upon entry.
- After the first successful move, Vecna gains 1 Health Point every second.
- A hidden Exit Door is randomly generated in the labyrinth.
- The player wins by reaching the Exit Door with more health than Vecna.
- The game ends if the player's health reaches zero.
